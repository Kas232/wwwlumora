# wwwlumora # Lumora | Beauty That Shines Naturally ✨

Lumora is a modern, responsive skincare e‑commerce website built with **HTML, CSS, and JavaScript**.  
It showcases premium products, an interactive cart system, and a functional contact form.

## 🚀 Features
- **Responsive Navbar** with smooth scrolling
- **Hero Section** highlighting brand message
- **Product Grid** with Add to Cart functionality
- **Cart System** (items saved using localStorage)
- **Contact Form** with validation and confirmation
- **Modern UI** using Playfair Display & Poppins fonts
- **Responsive Design** for mobile and desktop

## 🛠️ Tech Stack
- **HTML5** for structure
- **CSS3** for styling (flexbox, grid, media queries)
- **JavaScript (ES6)** for interactivity
- **LocalStorage** for cart persistence
